# Projeto: Mural de Avisos para WhatsApp (Case UX/UI)

## Contexto
Atividade acadêmica para a disciplina de Interface e Jornada do Usuário (ADS - UDF). O desafio consistia em integrar uma nova funcionalidade a um aplicativo existente, focando em redução de fricção e coerência visual.

## A Solução
Desenvolvi o **Mural de Avisos**, uma funcionalidade voltada para grupos ativos (ex: turmas de faculdade) onde informações críticas são frequentemente perdidas no fluxo de mensagens.

### O que foi feito:
* **Tela 1:** Interface clássica do WhatsApp (Ponto de partida).
* **Tela 2:** Menu de contexto com a nova opção "Mural do Grupo".
* **Tela 3:** Formulário simplificado para fixação de novos avisos.
* **Tela 4:** Visualização centralizada dos avisos fixados (RSVP/Acompanhamento).

## Decisões de Design (UX/UI)
* **Consistência:** Uso do Design System do Android/WhatsApp (Cores: #E5DDD5 e #DCF8C6).
* **Arquitetura:** Inserção no menu de "três pontinhos" para manter a interface principal limpa, atendendo ao perfil de usuário tecnologicamente proficiente.
* **Acessibilidade:** Botões com raio de canto (Corner Radius) de 25px para facilitar o clique e melhorar a estética.
